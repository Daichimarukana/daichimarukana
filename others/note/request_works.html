受け入れが可能なご依頼について

- 静的Webサイトの制作

以上のご依頼内容が現在お受付可能なものとなっております。
動画編集のお仕事などはお受付しておりません。
静的サイトというのはSNSのようなすぐに情報が更新されるものではなく、ごく一般的なHTMLの編集を手作業でして更新をしていただくようなもので、私が制作をお受け付けしているのはその中でもHTML、CSS、JavaScriptのみを使用したものとなっております。

簡単にどんなサイトをお作りすることができるかを挙げますとこのdaichimarukana.comや私の運営しているサービス一覧ページにある"uwuzu"を除いた配布サイトなどです。

どのようなジャンルのサイトでも自分の能力で可能そうであればご依頼はお受付しようと思っております！
個人のプロフィールサイトやグループのオフィシャルサイト、イベントのホームページや会社のホームページなど、広く受け付けたいです！

サイトのサンプル

実際に私が過去に作成したサイトのスクリーンショットをここに置いておきます！
私が作成したアプリの配布ページなどあまり参考にならないかもしれませんがご了承ください。

お作りできるサイトについて

重要 : 私がお作りできるのはサイト内のファイルです(HTMLやCSS、JSなど)
サーバー(サイトを公開するための機器)の用意やドメインの取得作業などは依頼者様側でお願いしております。

- 商用利用 : OK
- 二次利用・再配布 : 要相談
- サイトの対応ブラウザ : 原則Chrome 120以降(IE11以前に対応させる場合は要相談)
- ご依頼者様側からのサイトのイメージラフ : 何枚でも可(完成後のイメージを提出してくださる場合はpng,jpg,webp,gif形式でお願いします)
- こちらからのラフ : なし(作成しながらちょくちょくイメージをお伺いすることはあります)
- アダルトサイト : 不可
- ネットショップ : 不可
- 違法サイト : 不可
- phpやNode.jsなどを使用した動的サイト : 不可
- ジャンル : 要相談
- こちらからのラフ提案数 : 無制限
- 完成後の無料修正回数 : 3回
- お渡しする時のファイル形式 : zip形式
- 画像や動画のサイト内表示 : 可(合計ファイルサイズは5GBまででお願いします)
- サーバーの用意とネット上への公開作業 : 不可
- スマホ対応 : 要相談
- フォントの使用 : 可(GoogleFontsなどを使用するかご依頼者側で用意していただくなどの選択肢となっております。)

納期について

納期は原則0日~1ヶ月ほどを想定しております。
お作りするサイトの複雑さにもよりますが1~2ページ程なら1週間ほど、それ以上なら5日~1ヶ月ほどかと思われます。
こちらの都合によりお作りできない・制作を中止されていただく・納品までが長くなってしまう場合こちらより連絡させていただきます。
大変ご迷惑とは思われますがご了承ください。

料金表です。
ここに載っていないオプションや独自の機能などを使用したい場合は相談してください。
完成後のサイト更新等はご依頼者様側でお願いしております。

※3:jQueryを使用することで私がJavaScriptを書く時の負担が軽減されます。許可していただけるとありがたいです。

納品までの流れ

1. ご依頼者様からのご依頼
まずTwitterのDMかメールにて依頼したい旨をお知らせください。

2. 依頼内容の送信
私に依頼したい内容について送信していただきます。
このページの最下部にある依頼テンプレートにご記入いただき送信していただけるとありがたいです！
また、ご依頼者様からの完成後のイメージなどがありましたら添付していただけると大変ありがたいです！
もし質問などございましたら何度でも質問してください！

3. こちらからのご連絡
こちらから制作依頼を承るかどうかを返信させていただきます。
また、こちらからの質問などが含まれる場合もございます。

この工程でご依頼内容の詳細やお値段について、納期などについてお話させていただきます。

4. サイトの制作
ご依頼内容に合わせてサイトの制作を開始します！
もし途中で質問や追加での希望、修正や依頼を取りやめたいなどがありましたらご連絡ください。
なお、依頼の取りやめなどの場合は制作状況に応じてこちらよりお支払いをお願いする場合がございます。
ご迷惑とは思われますがご了承ください。

5. 修正
制作したサイトの修正を行います。
3回目まで無料でお受付いたします！
なお、ゼロから作り直すなどはお受付できませんのでご了承ください。
もし修正点などがなければそのままお支払い、納品へと進みます。

6. お支払い
お支払いをしていただきます！
現在はBOOTHでのお支払いのみをお受け付けしております。
もし支払いが遅れてしまう場合は事前にご連絡ください。
修正が完了し次第BOOTHの商品ページのリンクをお送りします。

6. 納品
お支払後、お支払いをこちらから確認させていただきます。
BOOTHでの購入が完了すると商品のダウンロードが可能になるかと思われます。
可能であればダウンロードしていただき商品の中身のスクリーンショットなどを撮っていただき送っていただけると幸いです。

金額ぴったりのお支払を確認させていただき次第GigaFile便でサイトを構成するファイル(HTML,CSS,JSなど)をzip形式にまとめて送信させていただきます。
GigaFile便が使用できない場合は事前にご連絡ください！
なお、ダウンロード期限は14日、ダウンロードパスワードなどを設定させていただきます。
このダウンロードファイルの中身が完成したサイトの中身です。
ご自身でWebサーバーにアップロードすることでサイトのインターネット上への公開が可能です！

納品の時点ではインターネット上にサイトは公開されていません。サーバーへのアップロードなど公開作業は一切こちらでは行いません。

サイトの更新について

サイトの更新作業もご依頼者側で行っていただきます。
ある程度HTMLをさわれると更新作業が行えるかと思われます！
こちらでの更新作業は一切おこないませんのでご了承ください。

Webサーバーとは

Webサーバーへのアップロードなどは完全に私の行える範囲外なのでサポートなどは行えませんが、一応軽く記載しておきます！

Webサポートとは、サイトをインターネット上へ公開するためのコンピューターを指す言葉です。
ロリポップさんやさくらのレンタルサーバー、エックスサーバーやコアサーバーなどがレンタルサーバーとして使いやすいかと思われます。
Apacheやnginxなどを使用し、独自サーバーを用意することも可能ではありますが、専門的な知識が必要となるためある程度知識のある方が行うと良いと思われます。

Webサーバーに納品したzipファイルの中身をアップロードし、ドメイン(インターネット上の住所)の設定などを行うことでインターネット上への公開が可能です。
この納品後インターネット上へ公開する作業に関してこちらでのサポートは一切しません！
