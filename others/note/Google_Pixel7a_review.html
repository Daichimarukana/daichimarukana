Pixel 7aを買ったおはなし

更新日

2024/02/18

筆者

だいちまる

はじめに

ということでおニューのスマホを手に入れたので本音でレビューしてみたというやつです。
少々厳しい言葉などが含まれているかもしれませんがあくまでも個人の感想ですのでご了承ください。
結論から申し上げますと、買う価値ありです。

開封

まず箱からですが正面にドドンとぴくせるって書いてあります。
他のスマホの箱と大きく違うことはないと思います。

箱を開けるとすぐにPixel 7aがでてきます。
同梱品は Google Pixel 7a、クイックスイッチアダプター(USB Type-AとType-Cを変換するやつ)、充電ケーブル(両端C端子)、SIMピン、Felicaの場所書いてる青いやつ、安全&保証&規制について書いてる説明書 です。
充電器やケース、フィルムなどはついていないので別途用意する必要があります。
Googleのシールとかあったら欲しかったな...

外観

背面は叩くとプラとガラスの中間みたいな音のなる素材でできていて、パッと見はガラスです。(なんか複合素材らしい)
中央にGoogleの"G"ロゴ、丈夫に少しの余白を残してバー上のカメラ、フラッシュライト部があります。
カメラの部分は2眼がくっついているデザインです。

表面は一面画面で6.1インチ、若干他のスマホと比べるとベゼルが太いですが気にならない程度のベゼルがあります。
画面下部はすこ～しだけベゼルが広いです。 インカメラはパンチホール式で画面内指紋認証があります。
ゴリラガラス クラス3らしいです。

側面は左側にSIMスロット、
上部にマイク穴...？(わからん)みたいなもの、
右側に上から電源、音量(ここが他の端末を逆)
下部にはスピーカー、マイク、USB Type-C端子があります。
アルミっぽくてさらさらしてる材質です。MicroSDは使えません。

完全に個人的な感想ですが、結構重たくて分厚いです(そんなに気にするほどではない)
重さはケースやガラスフィルムをつけると200gを超え、厚さはケースありで1.2cmほど
少し分厚くて存在感があります。
全体的な大きさはとても大きくも小さくない感じです(iPhone 4sと比べると大きいです)

画面とか

有機EL(OLED)で90Hz表示に対応しています。
前使っていたMotorola Edge 20くんと比べて少し色が落ち着いていますが、これといって薄いとかいうわけではなく困ることはありません。(Moto Edge 20くんが過度に鮮やかだった説)
また、外で使っても明るさ等で困ることはありません。

スペック～

今更かもしれませんがスペック

SoC : Google Tensor G2
RAM : 8GB
ストレージ : 128GB
画面サイズ : 6.1インチ
本体サイズ : ぼくのおてての80ぱーせんとくらい

ディスプレイ解像度 : 2400x1080らしい きれい
防水&防塵 : IP67らしい(水没させてないからわかりません)
ラジオ・ワンセグ : ない
GPS(位置情報) : GPS(アメリカ)、GLONASS(ロシア)、Galileo(EU)、QZSS(日本のみちびきさん)、BeiDou(中国)
Wi-Fi : Wi-Fi 6E（802.11ax）
NFC : ある。Felica対応。

充電端子 : USB Type-C
カメラ : きれい(64MPの方でとると16MPくらいになります(4画素を1画素と扱うため))

カメラ

普通に結構きれいです。
ほぼ参考にならない作例はこんな感じ↓

すごいのが補正機能でこんな感じでブレを減らしたり物を消したり𝘿𝙮𝙣𝙖𝙢𝙞𝙘にできます。

性能面

普通に使う分には快適です～！
激重3Dゲームなどをしない分には十分だと思います。
Antutuスコアは自分の環境で約83万点でした！
今のところ不満はありません。

良かった点

まず価格と性能からみてコスパが良い。
決して安いものでは無いのですが性能から見るととても優秀かと思います。
また、素のAndroidを搭載しているのも良いところで、これは個人個人意見が分かれると思いますが自分は素のAndroidが好きなので嬉しいポイントです。
他にもアップデートがすぐに来る、アップデート期間が長い、カメラの画質が比較的良い、Felicaに対応しているところなどが良かった点です！

値段的には優秀かと、買ってよかったな～と思います！！

悪かった点

まず画面内指紋認証が本当にクソ。
はっきり言わせてください。マジでクソです。使い物にならない。
指紋認証の登録などを試行錯誤したり、一度登録した指で失敗しまくっても削除せずに認証を続ければPixel側が学習したのか自分の場合一週間ほどで使える程度に指紋認証が通るようになったのですが、それでも認証精度は悪いです。
登録してすぐの頃は9割が失敗するほどです。自分の指紋が無いんじゃないかと疑うレベルでした。
「顔認証でも大丈夫！」という方は大丈夫かもしれませんが、指紋認証派の方にはおすすめできません。

また、Pixelランチャーの使い勝手もあまり良いものとは言えません。
アプリを縦における数が少なかったりスナップショットというウィジェット的なものが消せなかったり、アプリアイコンの形を丸形から変更できなかったり慣れるまで不満点は多いと思います。
個人的に気になったのはiPhoneのようにインストールされたアプリを横に並べられないこと。
インストールしたアプリをランチャーの最も右のページに自動で置いてくれる機能はあるのですが、下からスワイプしたときにでるドロワーを無効にすることはできません。
ドロワーでのアプリ並び替えや通知バッジの通知数の表示もできないのでiPhoneやカスタムされたAndroidから移行しようと考えている方には要注意です。

他には本体右側面の電源と音量ボタンが流通している多くのAndroidスマホと比べて上下逆についていることやフラッシュライトが(前使っていた機種と比べて)若干暗いところなどがあります。

総合評価

総合評価 : ★★★★☆
良いです。
はじめてスマホを買う方、iPhoneじゃなくても問題ない方、素のAndroidが搭載されている機種から乗り換える方にはおすすめです！
ガチで激重ゲームをしようと考えている方、超ハイスペ、超高画質カメラなどを求める方にはおすすめできません。
課題点としては画面内指紋認証がかなり影響してると思うので側面指紋認証や背面指紋認証のスマホを検討するのも一つの手かと思います。

結論 : 買う価値あり！

おわりっ！
