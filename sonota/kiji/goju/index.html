ダウンロード前の注意

・事前にPCにMeCab用IPA辞書がインストールされている必要があります。
　Binary package for MS-Windowsをインストールすることを推奨します。
　インストール時、文字コードはShift-JISに指定して下さい。

・Windows10以降でないと動作しません。

・Windows環境でのみ動作します。mac os、iOS、Linux等は非対応です。

・インターネットに接続されていると快適に動作します。(オフラインでも
　一応動作します)

・地域を指定して天気を聞く際に、Geocoding API version 1.2に接続します
　ので追加でのダウンロードが少々ございます。

・初期設定時にユーザー名・誕生日を設定しますが、これらの情報はごじゅ
　の対応と、誕生日のお祝いのときに使用します。

・ごじゅの回答時、回答文のバイト数が奇数の場合、文字化けが起こる可能
　性がございます。

・ごじゅは音声認識、音声を使った対話は出来ません。

・ごじゅに追加してほしい機能等ございましたら「フィードバック」と入力して送信してください。
